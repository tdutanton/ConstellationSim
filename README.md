# Спутниковая группировка — демо-симулятор

Java-проект, демонстрирующий работу с абстрактными спутниками и их группировками.

## Описание

Проект моделирует:

- абстрактный спутник (`Satellite`);
- два типа спутников:
    - **Связной** (`CommunicationSatellite`) — передаёт данные;
    - **Дистанционного зондирования Земли** (`ImagingSatellite`) — делает снимки;
- группировку спутников (`SatelliteConstellation`), которая может активировать спутники и запускать выполнение миссий.

Основная логика запускается из класса `Main`.

## Возможности

- Создание и активация спутников (при достаточном заряде батареи);
- Выполнение миссий с учётом расхода энергии;
- Автоматическая деактивация при разряде батареи;
- Вывод подробной информации в консоль (опционально).

## Запуск

Для отображения unicode символов рекомендуется запускать непосредственно в IDE (например ItellijIDEA).

Или напрямую в консоли (возможна автозамена unicode символов на "?"), находясь в папке src:

```bash
javac *.java
java Main
```

Также можно запустить уже созданный jar архив:

```bash
java -jar .\executable\ConstellationSim.jar
```

Для проверки различных состояний спутников рекомендуется:

- запускать программу несколько раз (спутники генерируются со случайным значением заряда батареи)
  ИЛИ
- в Main.java указать параметры в конструкторах спутников с необходимым зарядом батарей.

## Документация

Для создания документации:

```bash
javadoc -d docs -encoding UTF-8 -docencoding UTF-8 -charset UTF-8 ./src/*.java
```

После создания документация будет расположена в **docs/index.html**
